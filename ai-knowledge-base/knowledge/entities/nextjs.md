# Next.js

## What it is
React-based full-stack framework used for the Beldify frontend. Version 15 is used in this project.

## Key facts
- Version: Next.js 15
- Uses App Router architecture
- Frontend source in `beldify-frontend/src/`
- Build commands: `npm run build:dev`, `npm run build:prod`
- Dev server: `npm run dev` on port 3000
