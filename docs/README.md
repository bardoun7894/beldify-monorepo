# Documentation

Project documentation for Beldify platform.

## Structure

| Directory | Purpose |
|-----------|---------|
| `api/` | API endpoint documentation |
| `guides/` | How-to guides and tutorials |
| `architecture/` | System design decisions and diagrams |
| `changelog/` | Version history and migration notes |

## Contributing

When adding documentation:

1. **Use the correct directory** based on content type
2. **Follow the template** from `CLAUDE.md`
3. **Include examples** that are copy-paste ready
4. **Link related docs** at the bottom of each file
5. **Update this README** if adding new sections
